# QR Code Generator

Bu Python uygulaması, metin veya URL'leri QR kodlarına dönüştürmenizi sağlar.

## Özellikler
- Metin/URL'den QR kod oluşturma
- QR kodunu PNG formatında kaydetme
- Basit ve kullanıcı dostu arayüz

## Gereksinimler
- Python 3.x
- qrcode
- Pillow

## Kurulum
```bash
pip install -r requirements.txt
```

## Kullanım
```bash
python qr_generator.py
```
